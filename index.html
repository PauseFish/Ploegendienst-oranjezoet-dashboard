⚙ Instellingen

Waarom een GitHub token?
De instellingen (e-mail + drempel) worden opgeslagen in config.json in je GitHub-repository. De scraper leest dit bestand elke 30 minuten.

Om via deze pagina direct te kunnen opslaan heb je een Personal Access Token nodig:
1. Ga naar GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens
2. Klik Generate new token
3. Stel onder Repository permissions → Contents in op Read and write
4. Plak het token hieronder — het wordt alleen lokaal opgeslagen in je browser.
Geen token? Download config.json en upload het handmatig via GitHub → jouw repo → config.json → het potlood-icoon.
